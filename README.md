## 📖 학습 방식

<br/>

### 01 · 간격 반복 — SRS 기반 복습

**SRS(Spaced Repetition System)** 는 '기억 곡선' 이론을 기반으로,  
사람이 단어를 잊어버릴 타이밍에 맞춰 복습 시기를 자동으로 조절하는 학습 방법입니다.

> 💡 잘 아는 단어는 복습 간격을 늘리고 — 헷갈리는 단어는 자주 반복시켜 줍니다.

- 퀴즈 결과(정답 / 오답)에 따라 **다음 복습 시기가 자동으로 조정**됩니다.
- **오늘의 복습** 옵션을 켜면 오늘 복습이 필요한 단어만 골라 학습할 수 있어, 학습 시간을 절약합니다.

<br/>

### 02 · 퀴즈

| 유형 | 설명 |
|:---:|:---|
| 🔤 **객관식 퀴즈** | 보기 중 정답을 고르는 방식 |
| 🃏 **카드 맞추기** | 단어와 뜻을 짝지어 맞추는 방식 |

정답·오답 결과는 **즉시 학습 기록에 반영**되어, 나만의 맞춤 학습 환경이 자동으로 구성됩니다.

<br/>

### 03 · 뜻 숨기기

뜻을 가리고 **스펠링만 보면서 의미를 떠올리는 회상 학습**을 할 수 있습니다.  
단순히 단어를 눈으로 읽는 것보다 훨씬 강력하게 기억에 남습니다.

<br/>

### 04 · 아는 단어 숨기기

이미 외운 단어는 목록에서 숨겨,  
**아직 학습이 필요한 단어에만 집중**할 수 있습니다.

<br/>

### 05 · 별표 단어 ⭐

집중 학습이 필요한 단어를 ⭐로 표시하고,  
복습 기능과 연계해 해당 단어만 모아 반복 학습할 수 있습니다.

<br/>

### 06 · 범위 출력

전체 단어장에서 **원하는 구간(예: 1번 ~ 100번)만 지정**해 학습합니다.  
방대한 단어장도 단계별로 나눠 부담 없이 학습할 수 있습니다.

<br/>

### 07 · 셔플

단어 카드를 **무작위 순서로 섞어** 표시합니다.  
단어장 순서 암기로 인한 착각을 방지하고, 모든 단어를 골고루 학습하게 돕습니다.

## 🔊 발음 & 어휘 확장

<br/>

### 08 · TTS 음성 재생

단어 발음을 **TTS(Text-to-Speech)로 즉시 들을 수 있습니다.**  
반복 청취와 따라 말하기로 듣기·말하기 학습 효과를 높여보세요.

<br/>

### 09 · 국가별 자동 TTS 설정

단어장 파일에 저장된 **언어 태그(BCP 47)를 자동 인식**하여,
각 단어에 맞는 국가별 TTS 음성으로 자동 설정됩니다.

<br/>

<div>

| 🇺🇸 | 🇯🇵 | 🇨🇳 | 🇫🇷 | 🇩🇪 | 🇪🇸 | 🇮🇹 |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| English | 日本語 | 中文 | Français | Deutsch | Español | Italia |

</div>

> 💡 단어장에 언어 태그가 없으면, 기본적으로 **영어**가 설정됩니다.

<br/>

### 10 · 예문 표시 / 숨김

예문을 통해 **문장 속에서 단어가 실제로 어떻게 쓰이는지** 파악할 수 있습니다.  
단어의 뜻만 아니라 활용법까지 함께 익히세요.

<br/>

### 11 · Plus 단어

유의어·반의어·관련어 등 **확장 어휘 정보**를 함께 제공합니다.  
한 단어를 중심으로 어휘력을 폭넓게 키울 수 있습니다.

<br/>

### 12 · 사전 연동

**네이버 영어 사전**과 연동되어,  
표제어의 자세한 뜻·예문·발음 등을 바로 확인할 수 있습니다.

<br/>

### 13 · 실시간 검색

영어 스펠링 또는 한글 뜻의 **일부만 입력해도 즉시 결과를 필터링**해 보여줍니다.

### 15 · 내보내기 / 가져오기

학습 데이터를 **파일 또는 텍스트 형식**으로 내보내거나 가져올 수 있습니다.  
기기가 바뀌어도 학습 기록을 안전하게 이어갈 수 있습니다.

<br/>

### 16 · 메모

| 방식 | 설명 |
|:---:|:---|
| ⌨️ **타이핑 메모** | 키보드로 텍스트 작성 |
| ✏️ **드로잉 메모** | 손으로 직접 그리며 필기 |

- 단어장마다 **전용 메모 공간이 1개씩** 제공됩니다.
- 작성 · 저장 · 삭제를 모두 지원합니다.

### 20 · JSON 단어장 구조

단어장은 **JSON 파일 형식**으로 구성됩니다.
