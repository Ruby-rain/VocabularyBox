# 📦 VocabularyBox
### 반복 노출 + 간격 반복(SRS) + 퀴즈 + TTS로 완성한 개인 맞춤 단어 학습 환경

## 핵심 기능

### 1) 학습 방식
- **범위 출력**: 전체 단어장 중 원하는 구간을 지정해 학습할 수 있습니다.
- **셔플 기능**: 단어 카드 배열을 무작위로 출력하여 실전 감각을 향상시킵니다.
- **뜻 숨기기**: 뜻을 가리고 스펠링만 보고 의미를 떠올리는 회상 학습을 지원합니다.
- **퀴즈 기능**: 단어 의미를 선택하는 '객관식', 단어와 뜻을 맞추는 '카드 맞추기'를 제공합니다.  
  정답/오답은 즉시 DB에 저장되어 맞춤형 학습 경험을 구축하는 데 활용됩니다.
- **아는 단어 숨기기**: 이미 아는 단어를 숨김 처리하여 학습 효율을 높입니다.
- **별표 단어(★)**: 집중 학습이 필요한 단어를 별표로 표시하고, 별표 단어만 골라 복습할 수 있습니다.
- **간격 반복 SRS**: 퀴즈 결과에 따라 복습 시점을 자동 조정합니다.  
  학습일에 복습할 단어만 골라 복습할 수 있어 효율적입니다.

### 2) 발음 · 예문 · 확장 학습
- **TTS(Text-to-Speech)**: 단어 발음을 재생하여 듣고 따라 말하며 학습할 수 있습니다.
- **예문 표시/숨김**: 예문을 통해 문장 속 사용 방식을 빠르게 파악합니다.
- **Plus 단어**: 유의어, 반의어, 관련어 등 확장 정보로 어휘력을 넓힐 수 있습니다.
- **사전 기능**: 네이버 영어 사전으로 연동되어 표제어의 자세한 뜻/예문/발음 등을 확인할 수 있습니다.

### 3) 탐색 · 집중
- **실시간 검색**: 영어 스펠링 또는 한글 뜻 일부만 입력하여 결과를 즉시 필터링합니다.
- **스크롤 프로그레스바**: 상단 진행 게이지로 현재 위치를 시각적으로 확인합니다.

### 4) 메모 · 데이터 관리
- **메모 기능**: 타이핑 및 드로잉 방식으로 메모 작성/저장/삭제를 지원합니다.  
  각 단어장에는 하나의 메모 저장 공간이 제공됩니다.
- **내보내기/가져오기**: 학습 데이터를 파일 또는 텍스트로 내보내거나 가져오는 수동 동기화 기능을 제공합니다.
- **단어장 통계**: 별표 단어/아는 단어/틀린 단어/총 단어 수, 일별 학습 단어 수 그래프를 제공하며, 단어별 오답 횟수를 한눈에 확인할 수 있습니다.

---
## 데이터 구조
- 단어장은 **JSON 파일 기반**으로 구성되어 있어, 단어 수나 용량과 관계없이 다양한 단어장을 자유롭게 교체하며 사용할 수 있습니다.
- 학습 기록은 로컬 IndexedDB에 저장되며, 내보내기/가져오기로 수동 동기화를 지원합니다.

> ⚠️ 학습 데이터는 사용 중인 기기의 브라우저(로컬 저장소)에 저장하고 관리합니다.
> 브라우저 캐시/사이트 데이터를 삭제하면 학습 데이터가 함께 삭제될 수 있습니다.
> VocabularyBox는 별도의 보증을 제공하지 않으며, 삭제된 학습 데이터에 대해 책임지지 않습니다.

## UI · 테마
- **라이트/다크 모드** 전환으로 눈의 피로를 줄이고 집중력을 유지합니다.
- **파스텔톤 디자인**으로 가독성, 색상 대비, 여백이 안정적으로 정돈된 학습 환경을 제공합니다.
